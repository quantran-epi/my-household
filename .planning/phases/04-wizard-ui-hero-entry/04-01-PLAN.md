<interface_context>
The wizard container (built in 04-03) renders these step widgets and passes callbacks. Build the widgets to these exact prop signatures so the container imports them without change:

- `WizardIngredientStep`: props `{ onNext: (answer: Partial<WizardAnswers>) => void }`. Reads its initial value from `selectWizardAnswers().selectedIngredientIds`. Calls `onNext({ selectedIngredientIds: ids })` on advance and `onNext({ selectedIngredientIds: [] })` on skip.
- `WizardPreferenceStep`: props `{ onNext: (answer: Partial<WizardAnswers>) => void; onBack: () => void }`. Calls `onNext({ preferredTags: selectedTags })` on advance and `onNext({})` on skip.

`WizardAnswers` and `WizardPreferenceAnswers` are defined in `src/Store/Models/Wizard.ts` — import the type, do NOT redefine it.
</interface_context>
