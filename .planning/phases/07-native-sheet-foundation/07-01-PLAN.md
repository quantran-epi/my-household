<objective>
Extract the pure drag-decision logic for the native sheet into a standalone DOM-free module, backed by exhaustive Jest unit tests. This isolates the decision rules (when a drag starts, and whether a release dismisses or springs back) from any UI/gesture wiring, so the rules can be tested deterministically and reused.

Purpose: Establish a verified logic foundation (SHEET-01, SHEET-03, SHEET-04) before any UI integration. Pure functions are trivially testable and prevent regressions when the gesture layer is wired later.
Output: `src/Components/FastOverlay/dragDecision.ts` (two pure functions + one tunable constant) and `src/Components/FastOverlay/dragDecision.test.ts` (full branch + boundary coverage).
</objective>
